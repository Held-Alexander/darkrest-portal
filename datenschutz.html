1. Hosting (GitHub Pages)

Diese Seite wird über GitHub Pages (GitHub Inc., USA) bereitgestellt. GitHub erhebt technisch notwendige Daten (IP-Adresse, Browser-Informationen) in Logfiles. Da die Daten in die USA übertragen werden können, weisen wir darauf hin, dass GitHub angemessene Sicherheitsstandards einhält.

2. Cookies & Tracking

Wir verwenden keine Tracking-Cookies und keine Analysetools. Diese Website nutzt lediglich technisch notwendige Daten für die Auslieferung der Inhalte.

3. Minecraft-Server Daten

Verarbeitet werden UUID, Username und IP zur Bereitstellung des Spielbetriebs. Protokolldateien werden nach 7 Tagen automatisch gelöscht.

4. Externe Links (Discord & YouTube)

Wir verlinken auf Discord und YouTube. Sobald du einen dieser Links anklickst, verlässt du unsere Website. Ab diesem Zeitpunkt gelten die Datenschutzrichtlinien des jeweiligen Anbieters (Discord Inc. bzw. Google Ireland Limited).

Du kannst jederzeit die Löschung deiner spielbezogenen Daten verlangen.
